## 📧 Contact Form Setup (EmailJS)

This project uses **EmailJS** for sending messages directly from the contact form.

### Install EmailJS

```bash
npm install @emailjs/browser
```

### Configure

Replace the following values inside `Contact.jsx`:

```js
emailjs.send(
  "YOUR_SERVICE_ID",
  "YOUR_TEMPLATE_ID",
  form,
  "YOUR_PUBLIC_KEY"
);
```
